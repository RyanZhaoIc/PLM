# Part-level Multi-labeling

This is the implementation of the paper: Estimating Noisy Class Posterior with Part-level Labels for Noisy Label Learning (CVPR 2024)
## Requirements
Python 3.8.13\
Pytorch 1.10.0\
torchvision 0.11.0\
cudatoolkit 11.3.1\
easydict 1.9\
scikit-learn 1.0.2

## Training
### Demo for CIFAR10
	python lnl.py --ds cifar-10
